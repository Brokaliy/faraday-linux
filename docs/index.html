Security

Hardened at every layer

Every decision is documented. Every feature can be toggled.

󰦝

Tor by Default

All TCP is transparently proxied through Tor via nftables. No configuration required. Apps need zero changes.

Hardened Kernel

linux-hardened with KASLR, stack canaries, restricted ptrace, hardened usercopy, and full sysctl paranoia.

LUKS2 Encryption

Full disk encryption enforced by the installer. Argon2id key derivation resists GPU brute-force. No plaintext swap.

󰩠

Encrypted DNS

DNS routed through Tor's DNSPort — no DNS leaks. dnscrypt-proxy as a fallback layer with DoH.

󰛵

MAC Randomization

Network interface MAC randomized on every connection. Prevents cross-network tracking.

AppArmor

Mandatory access control confines every app. Compromised processes can't escape their sandbox.

USBGuard

Unknown USB devices blocked by default. Stops BadUSB and rubber-ducky attacks cold.

Camera & Mic Off

Webcam and microphone drivers blacklisted at kernel level. You choose when to enable them.

󰖂

Mullvad VPN

Optional second layer. No-logs, accepts cash and Monero, RAM-only servers. Works over Tor.

Desktop

Modern. Minimal. Actually usable.

Built on Hyprland — Wayland only, no X11 side-channels. Looks good without sacrificing security.

⬡

Hyprland + Waybar

Tiling compositor with animated borders, blur, and a Tor-status indicator in the bar.

Kitty Terminal

GPU-accelerated, ligatures, 0.95 opacity with blur. JetBrainsMono Nerd Font throughout.

Hardened Firefox

WebRTC off. SOCKS5→Tor locked via policy. uBlock Origin force-installed. No telemetry.

󰦝

Privacy Apps

Signal, KeePassXC, Mullvad, Tor Browser, mat2 metadata stripper — all preinstalled.

Fish + Starship

No shell history. Tor/VPN/DNS status in the prompt. fastfetch on every new terminal.

Calamares Installer

Custom branded installer with LUKS enforced. Each screen explains what a privacy feature does.

Install

Three steps to private.

01

Download the ISO

Grab the latest ISO from GitHub Releases. Built automatically by GitHub Actions on every commit.

github.com/Brokaliy/faraday-linux/releases →
02

Write to USB

Flash the ISO to a USB drive. Works with any 4GB+ drive.

sudo dd if=faraday-linux.iso of=/dev/sdX bs=4M status=progress
03

Boot & Install

Boot from USB. Use the live desktop to try it first, then launch the installer from the desktop. LUKS passphrase required — no way around it.

Testing in a VM? VMware or VirtualBox work fine. Set RAM to 4GB+, enable 3D acceleration, boot the ISO.
