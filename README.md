# Faraday Linux

**Privacy-first NixOS. Tails meets modern Hyprland.**

All traffic through Tor. LUKS encryption. Hardened kernel. A desktop normal humans can actually use.

---

## What it is

Faraday Linux is a NixOS-based privacy OS that builds into a bootable ISO.
Named after the Faraday cage — a mesh that blocks electromagnetic fields — Faraday Linux is a mesh that blocks surveillance.

**Security features:**
- `linux-hardened` kernel with full hardening patch set
- All traffic transparently proxied through Tor (no configuration needed)
- MAC address randomized on every network connection
- DNS routed through Tor's DNSPort — no DNS leaks
- Mullvad VPN preinstalled for an optional extra layer
- LUKS2 full-disk encryption enforced by the installer (Argon2id key derivation)
- Encrypted swap, or no swap (your choice — no plaintext RAM pages on disk)
- Firewall blocks all inbound traffic by default
- IPv6 disabled (IPv6 bypasses Tor by default)
- AppArmor mandatory access control
- USBGuard (unknown USB devices are blocked)
- Webcam and microphone disabled at kernel level (blacklisted modules)
- Hardened sysctl: ASLR max, ptrace restricted, ICMP blocked, SYN cookies on

**Desktop:**
- Hyprland (Wayland only — no X11 side-channels)
- Waybar with Tor status indicator
- Wofi launcher, Dunst notifications, Hyprlock screen locker
- Dark theme, cyan accents, Faraday cage wallpaper
- Firefox hardened (WebRTC off, proxied through Tor, uBlock Origin baked in)
- Signal, KeePassXC, Mullvad, kitty terminal, fastfetch with Faraday branding

## Quick Start

### Prerequisites

- Linux host with Nix installed
- Flakes enabled in `~/.config/nix/nix.conf`:
  ```
  experimental-features = nix-command flakes
  ```
- ~8GB free disk space, ~4GB RAM

### Build

```bash
git clone https://github.com/yourusername/faraday-linux
cd faraday-linux

# Build ISO (first build ~30-60 min; subsequent builds fast via cache)
./build.sh

# Test in VM first
./build.sh run

# Write to USB
sudo dd if=result/iso/faraday-linux-*.iso of=/dev/sdX bs=4M status=progress
```

### Install

Boot from the USB. The Calamares installer launches from the desktop:
1. Choose language/locale
2. Choose disk → full-disk erase with LUKS2 encryption is pre-selected
3. Set passphrase (12+ chars required)
4. Set username/password
5. Install (~10 minutes)

## Project Structure

```
faraday-linux/
├── flake.nix              — Nix flake (inputs, ISO/VM/installed outputs)
├── build.sh               — Build ISO, VM, or run QEMU for testing
├── home.nix               — Home Manager: Hyprland, Waybar, Wofi, Kitty, etc.
├── modules/
│   ├── hardening.nix      — Kernel, sysctl, AppArmor, USBGuard, memory hardening
│   ├── networking.nix     — Tor transparent proxy, MAC randomization, DNS, firewall
│   ├── desktop.nix        — Hyprland, PipeWire, fonts, portals, env vars
│   ├── packages.nix       — Privacy apps, Firefox hardening, ZSH config
│   ├── installer.nix      — Calamares config, LUKS enforcement, post-install
│   └── branding.nix       — OS identity, Plymouth, fastfetch, MOTD, faraday-status
└── assets/
    ├── wallpaper.png       — Faraday cage wallpaper (1920x1080)
    ├── logo.svg            — Faraday logo
    └── calamares/          — Installer branding (generated by build.sh if missing)
```

## Checking Your Security

```bash
# Terminal dashboard
faraday-status

# Verify Tor is working
torify curl https://check.torproject.org/api/ip

# Check your current IP (should be a Tor exit node)
torify curl https://api.ipify.org

# Check MAC randomization
ip link show
```

## Toggling Features

### Enable Webcam

```bash
# Load webcam driver (live session / current boot only)
sudo modprobe uvcvideo

# To permanently enable, add to configuration.nix:
# boot.blacklistedKernelModules = lib.filter (m: m != "uvcvideo") [ ... ];
```

### Enable Microphone

```bash
sudo modprobe snd_usb_audio
```

### Use Mullvad VPN (over Tor)

```bash
mullvad account login YOUR_ACCOUNT_NUMBER
mullvad connect
# Now you have: Your machine → Tor → Mullvad → Internet
```

### Disable Tor (not recommended)

```bash
sudo systemctl stop tor
# WARNING: Without Tor, your real IP is exposed.
```

## Threat Model

Faraday Linux is designed for:
- Journalists, activists, and whistleblowers on hostile networks
- Anyone who needs a clean, untraceable browsing session
- Security researchers who want a known-hardened baseline

Faraday Linux is **not** designed to protect against:
- Physical access attacks (use full-disk passphrase + keep the machine)
- Adversaries who control your Tor exit node + the destination (correlation attacks)
- Browser exploits that run code before traffic reaches Tor

For maximum anonymity, use the preinstalled **Tor Browser** rather than Firefox.
